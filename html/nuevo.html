Primero... ¿Sabes cómo funciona MARKDOWN?

Markdown es un lenguaje que permite dar formato a texto de forma sencilla. Utiliza caracteres especiales para estructurar documentos, como # para títulos, * para cursivas, ** para negritas y listas con - o números.
